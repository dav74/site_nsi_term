### exercice 4.1

Soit le programme Python suivant :

```python
def fct(n):
   if n > 5:
       return 1
   else :
       return n+fct(n+1)
a = fct(2)
```

Que vaut la variable a après l’exécution de ce programme (justifiez votre réponse) ?
